Frequently Asked Questions

How do I select a MIDI input source?

Open Settings from the menu bar icon and select your MIDI device from the Input dropdown. MSC2OSC will automatically reconnect if the device is disconnected and reconnected.

How do I configure the OSC destination?

In Settings, enter the IP address and port of your OSC server. The default is 127.0.0.1:8000 for localhost.

How do I customize the OSC path format?

Use tokens like {command}, {cue}, {list}, {path}, {format}, and {device} in your OSC path template. For example: /msc/{command}/{list}/{cue}

What MSC commands are supported?

MSC2OSC supports all standard MSC commands including GO, STOP, RESUME, TIMED_GO, LOAD, SET, FIRE, ALL_OFF, RESTORE, RESET, GO_OFF, and more.

Can I filter specific commands?

Yes, you can enable or disable forwarding for individual MSC commands in the Settings panel.

What macOS versions are supported?

MSC2OSC requires macOS 14.0 (Sonoma) or later and runs natively on both Apple Silicon and Intel Macs.
