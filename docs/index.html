MSC2OSC

A lightweight macOS menu bar app that bridges MIDI Show Control to Open Sound Control in real-time.

Download for macOS

Features

Real-Time Translation

Sub-millisecond latency conversion from MSC to OSC messages.

Menu Bar App

Runs quietly in your menu bar, always ready when you need it.

Flexible OSC Paths

Customize OSC paths with tokens like {command}, {cue}, {list}, and more.

Command Filtering

Enable or disable specific MSC commands to forward only what you need.

Auto Reconnect

Automatically reconnects to MIDI devices when they're plugged back in.

Message Monitor

View incoming MSC and outgoing OSC messages with search filtering.

Quick Setup

1

Select MIDI Input

Choose your MSC source from the Settings menu.

2

Configure OSC Destination

Enter the IP address and port of your OSC server.

3

Customize Path Format

Optionally customize the OSC path template to match your setup.
